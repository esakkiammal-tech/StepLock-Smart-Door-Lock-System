# StepLock-Smart-Door-Lock-System

## Description
This project implements a smart door lock system using a step motor
controlled by two push buttons. One button locks the door and the other
unlocks it by rotating the motor in forward and reverse directions.

## Components Used
- Arduino UNO / Nano
- Step Motor
- Motor Driver IC
- Push Buttons
- Jumper Wires

## Working
- Press Lock button → Motor rotates forward
- Press Unlock button → Motor rotates backward

## Applications
- Smart home systems
- Door security
- Automation projects
